# Reviewer Checklist

## Claim Hygiene

- [x] Claim scope constrained to constructive universality.
- [x] Unbounded-memory assumption stated explicitly.
- [x] Non-claims listed (no quantum entanglement/speedup claim).

## Theory

- [x] Main theorem statement present.
- [x] Gate/memory/composition lemmas documented.
- [x] Formal appendix included (`FORMAL_APPENDIX.md`).
- [ ] Independent formal proof audit complete.

## Experiments

- [x] Unit tests pass in CI.
- [x] Repro commands documented.
- [x] Robustness sweep includes CI and failure envelopes.
- [ ] Cross-platform replication complete.
- [ ] Hardware-in-the-loop replication complete.

## Artifacts

- [x] Reports and scripts versioned.
- [x] CI runs current stack, not legacy-only tests.
- [ ] DOI snapshot published.
